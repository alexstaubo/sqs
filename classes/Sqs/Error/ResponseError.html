Class Sqs::Error::ResponseError
In: lib/sqs/exceptions.rb
Parent: StandardError

All responses with a code between 300 and 599 that contain an body are wrapped in an ErrorResponse which contains an Error object. This Error class generates a custom exception with the name of the xml Error and its message. All such runtime generated exception classes descend from ResponseError and contain the ErrorResponse object so that all code that makes a request can rescue ResponseError and get access to the ErrorResponse.
